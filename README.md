# ماهیر — لندینگ‌پیج رسمی

لندینگ‌پیج پریمیوم برند **ماهیر**؛ آژانس رشد کسب‌وکار که کسب‌وکارهای کوچک را از صفر تا صد رشد می‌دهد
(برندینگ، طراحی، تولید محتوا، جذب کاربر و افزایش فروش).

ساخته‌شده با **Next.js 16 (App Router)** · **TypeScript** · **Tailwind CSS v4** · فونت **Vazirmatn**.

---

## ویژگی‌ها

- 🎨 طراحی مدرن، مینیمال و لوکس — مشکی عمیق + طلایی ملایم + سفید
- 🌙 حالت تاریک به‌صورت پیش‌فرض
- 🔁 کاملاً راست‌چین (RTL) و فارسی
- 📱 موبایل‌اول و کاملاً ریسپانسیو
- ✨ انیمیشن‌های نرم اسکرول با CSS + IntersectionObserver (بدون وابستگی سنگین)
- ⚡ صفحه کاملاً استاتیک (SSG) و آماده دیپلوی

---

## بخش‌های صفحه

Navbar چسبان · Hero (+ ویدیو معرفی) · آمار اعتمادساز · مشکل · راه‌حل ماهیر · خدمات · فرآیند کار ·
نمونه‌کارها · نظرات مشتریان · مدل همکاری و قیمت‌گذاری · ضمانت نتیجه · چرا ماهیر؟ · سوالات متداول (۸ سوال) ·
تماس · فوتر (راه‌های ارتباطی مستقیم)

**صفحه‌ها:** `/` · `/about` (درباره ما) · `/submit` (ثبت پروژه) · `/thank-you` · `/portfolio/[slug]` ·
`/privacy` (حریم خصوصی) · `/terms` (قوانین) — به‌همراه **دکمه شناور تماس/واتساپ/تلگرام** روی همه صفحات.

> اطلاعات تماس (شماره، واتساپ، تلگرام، ایمیل) به‌صورت نمونه در `src/data/contact.ts` متمرکز است؛
> مقادیر واقعی را همانجا جایگزین کنید تا در کل سایت اعمال شود.

## اجرا

```bash
npm install      # نصب وابستگی‌ها
npm run dev      # محیط توسعه روی http://localhost:3000
npm run build    # بیلد استاتیک → خروجی در پوشه‌ی out/
npm run lint     # بررسی لینت
```

> این پروژه به هیچ متغیر محیطی یا کلید API نیازی ندارد.

## دیپلوی استاتیک روی هاست معمولی

پروژه با `output: 'export'` تنظیم شده و کاملاً استاتیک است:

```bash
npm install
npm run build      # پوشه‌ی out/ ساخته می‌شود
```

سپس **کل محتوای پوشه‌ی `out/`** را در ریشه‌ی هاست (مثلاً `public_html`) آپلود کنید. همین.
هیچ سرور Node یا بک‌اندی لازم نیست.

- فرم «ثبت پروژه» بدون بک‌اند کار می‌کند و دو حالت دارد (تنظیم در `src/data/form.ts`):
  1. **پیش‌فرض (بدون تنظیم):** اطلاعات را به‌صورت پیام آماده در **واتساپ ماهیر** باز می‌کند
     (شماره در `src/data/contact.ts`). سایت از همان ابتدا عملیاتی است.
  2. **Formspree (پیشنهادی برای دریافت لید در ایمیل):** در [formspree.io](https://formspree.io)
     یک فرم رایگان بسازید و `FORMSPREE_ENDPOINT` را در `src/data/form.ts` جایگزین کنید.
     از آن پس فرم لیدها را به Formspree می‌فرستد و در صورت خطا، خودکار به واتساپ برمی‌گردد.
  در هر دو حالت، پس از ارسال کاربر به `/thank-you/` هدایت می‌شود.
- هر مسیر به‌صورت `پوشه/index.html` خروجی می‌گیرد (`trailingSlash: true`) تا روی هاست‌های ساده درست کار کند.
- برای صفحه‌ی ۴۰۴ سفارشی، فایل `out/404.html` تولید می‌شود (اکثر هاست‌ها آن را خودکار استفاده می‌کنند).

### نمونه‌کارها (کیس‌استادی)

- بخش **«پروژه‌های موفق ماهیر»** در صفحه اصلی (`#portfolio`) سه کیس‌استادی با نتایج عددی نمایش می‌دهد.
- هر کارت دکمه **«مشاهده جزئیات»** دارد که به صفحه اختصاصی `‎/portfolio/[slug]` می‌رود
  (چالش، راهکار، نتیجه و تصاویر).
- چون هنوز عکس واقعی نداریم، برای هر پروژه بلوک‌های تصویر Placeholder به‌همراه **توضیح دقیق و
  یک پرامپت آماده تولید عکس با هوش مصنوعی** قرار داده شده است. محتوای نمونه‌کارها در
  `src/data/portfolio.ts` قابل ویرایش است.

### مدل همکاری و قیمت‌گذاری

بخش **«مدل همکاری با ماهیر»** (`#pricing`) سه پکیج مقایسه‌ای نتیجه‌محور را نمایش می‌دهد
(شروع رشد / رشد حرفه‌ای — پیشنهادی / رشد کامل و تحویل) با درصد پیش‌پرداخت و شرایط پرداخت
باقیمانده، یک باکس اعتمادساز و CTA به `/submit`. قیمت عددی ثابت درج نشده و پکیج‌ها در
`src/components/Pricing.tsx` قابل ویرایش‌اند.

### سیستم ثبت پروژه

- صفحه **`/submit`** فرم کامل ثبت پروژه را نمایش می‌دهد (نام، موبایل، اینستاگرام، وب‌سایت،
  حوزه فعالیت، وضعیت فعلی، مشکل، هدف و بودجه) با **اعتبارسنجی سمت کلاینت و سرور**.
- بعد از ارسال موفق، کاربر به صفحه **`/thank-you`** هدایت می‌شود.
- فرم به Route Handler **`/api/submit`** ارسال می‌شود که داده را اعتبارسنجی و **سمت سرور
  `console.log`** می‌کند (شبیه‌سازی‌شده). برای ارسال واقعی، در `src/app/api/submit/route.ts`
  محل مشخص‌شده با `TODO` را به تلگرام / ایمیل / CRM وصل کنید.
- تمام دکمه‌های CTA اصلی سایت (Hero، نتایج، تماس، Navbar) به صفحه ثبت پروژه لینک شده‌اند.

## دیپلوی روی Vercel

1. مخزن را به [vercel.com](https://vercel.com) وصل کنید و **New Project** را بزنید.
2. تنظیمات پیش‌فرض کافی است (نیازی به Environment Variable نیست).
3. **Deploy** را بزنید.

### اتصال دامنه mahir.ir

در داشبورد Vercel → **Settings → Domains**، دامنه‌های `mahir.ir` و `www.mahir.ir` را اضافه کنید و
رکوردهای DNS پیشنهادی Vercel را در پنل دامنه‌تان ثبت کنید.
